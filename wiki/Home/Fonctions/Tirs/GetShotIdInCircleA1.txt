====== GetShotIdInCircleA1 ======

Arguments:
	1) x-coordinate
	2) y-coordinate
	3) radius
Returns:
	Array
Returns the object IDs of the bullets inside the given circle in an array.

Inside the player script, it will only return enemy bullet IDs, and vice versa.
