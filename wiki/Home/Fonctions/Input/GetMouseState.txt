====== GetMouseState ======

Arguments:
	1) mouse button
		 - MOUSE_LEFT: Left Mouse Button
		 - MOUSE_MIDDLE: Middle Mouse Button 
		 - MOUSE_RIGHT: RightMouse Button
Returns:
	Mouse State
Returns the given mouse button's current state.

Refer to the list at the top for list of Key States
