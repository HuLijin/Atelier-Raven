Key State List

KEY_FREE: the key is not pressed.
KEY_PUSH: the key has been pressed.
KEY_HOLD: the key is being held.
KEY_PULL: the key has been released.

Virtual Key List

VK_LEFT: move left	VK_RIGHT: move right
VK_UP: move up		VK_DOWN: move down
VK_SHOT: shot		VK_BOMB: bomb
VK_SLOWMOVE: focus
VK_USER1: user key 1	VK_USER2: user key 2
VK_OK: confirm		VK_CANCEL: cancel

Key List

KEY_0 ... KEY_9： 0-9 Keys
KEY_A ... KEY_Z： A-Z Keys
KEY_F1 ... KEY_F10： F1-F10 Keys
KEY_NUMPAD0 ... KEY_NUMPAD9： 0-9 on the Number Pad
KEY_INSERT： Insert
KEY_DELETE： Delete
KEY_BACK： Backspace
KEY_RETURN： Enter
KEY_LCONTROL： Left Ctrl key	KEY_RCONTROL： Right Ctrl key
KEY_LSHIFT： Left Shift	KEY_RSHIFT： Right Shift
KEY_LEFT： Left Arrow Key	KEY_RIGHT： Right Arrow Key
KEY_UP： Up Arrow Key	KEY_DOWN： Down Arrow Key
