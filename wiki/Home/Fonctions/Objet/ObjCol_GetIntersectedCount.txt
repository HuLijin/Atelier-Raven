====== ObjCol GetIntersectedCount ======

Parameters:
 1) object ID (real)
Returns the number of times the specified object intersected with other objects during the previous frame.

For example, this can be used to determine how many player shots have collided with an enemy (similar to GetHitCount in v0.12m).
