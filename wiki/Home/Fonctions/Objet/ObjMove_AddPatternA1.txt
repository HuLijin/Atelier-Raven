====== ObjMove AddPatternA1 ======

Arguments:
 1) object ID (real)
 2) frame (real)
 3) speed (real)
 4) angle (real)
After the specified number of frames, changes the speed and angle of the object. NO_CHANGE can be used to preserve the original speed or angle.
