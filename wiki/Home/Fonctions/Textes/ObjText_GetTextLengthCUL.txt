====== ObjText GetTextLengthCUL ======

Arguments:
 1) object ID (real)

Returns: lengths (real array)
Returns an array of the lengths of each line of the specified text object. Newlines and ruby text are not counted. When using Japanese characters, both half-width and full-width characters count as 1 character.
