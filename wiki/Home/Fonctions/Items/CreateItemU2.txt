====== CreateItemU2 ======

Arguments:
	1) Item ID
	2) x-coordinate
	3) y-coordinate
	4) x-coordinate destination
	5) y-coordinate destination
	6) scoring
Return Value
	Object ID
Creates an item much like CreateItemA2, however it will spawn a user-defined item associated with the specified ID.
