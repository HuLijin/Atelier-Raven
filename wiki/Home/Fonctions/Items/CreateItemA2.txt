====== CreateItemA2 ======

Arguments:
	1) type
	2) x-coordinate
	3) y-coordinate
	4) x-coordinate destination
	5) y-coordinate destination
	6) scoring
Return Value
	Object ID
Same as CreateItemA1, except now it will quickly move towards the specified coordinates before slowly falling down.
