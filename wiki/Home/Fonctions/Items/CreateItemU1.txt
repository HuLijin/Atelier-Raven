====== CreateItemU1 ======

Arguments:
	1) Item ID
	2) x-coordinate
	3) y-coordinate
	4) scoring
Return Value
	Object ID
Creates an item much like CreateItemA1, however it will spawn a user-defined item associated with the specified ID.
