====== 1 Avant de commencer ======

En premier lieu, asurez vous de connaitre la documentation des deux fonctions ''NotifyEvent'' et ''NotifyEventAll''.
Je remets ici leur signature :

'''
NotifyEvent( scriptID, event_type, value )
'''
	'''
	scriptID : ID du script qu'on veut notifier (voir plus loin comment on le récupère)
	event_type : notification prédéfinie (voir plus loin)
	value : le fameux argument à passer au script (voir plus loin)
		
	NotifyEventAll( event_type, value )
	'''


Pour  vous apprendre à utiliser les notifications d'événements, je vais vous présenter un cas concret d'utilisation : le spawn d'items à la mort d'un ennemi.


Note du traducteur : voir le tutoriel de Sparen sur le spawn d'items si vous vous demandez pourquoi passer par les événements est nécessaire pour arriver à faire spawn les objets, il l'explique très bien. Personnellement j'utilise une autre technique qui ne nécessite pas de discuter entre plusieurs scripts, et qui se passe très bien des événements du coup, mais qui est plus complexe à  comprendre quand on débute.
