====== ObjRender GetBlendType ======

Arguments:
 1) object ID (real)
Returns the blend type for the specified object. Available blend types are:

BLEND_ALPHA
BLEND_ADD_RGB
BLEND_MULTIPLY
BLEND_SUBTRACT
BLEND_ADD_ARGB
BLEND_INV_DESTRGB
