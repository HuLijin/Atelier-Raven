====== StartSlow ======

Arguments:
	1) Slow Target
		 - TARGET_ALL
	2) FPS Slow Value (0-60)
Creates a pseudo slow effect by forcing Danmakufu to run at the specified FPS value. There is currently only one target available, TARGET_ALL. Use StopSlow to stop this effect.
