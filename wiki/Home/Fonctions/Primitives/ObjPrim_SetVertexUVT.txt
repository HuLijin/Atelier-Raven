====== ObjPrim SetVertexUVT ======

Arguments:
 1) object ID (real)
 2) vertex index (real)
 3) texture x-coordinate (real)
 4) texture y-coordinate (real)
Sets the UV-coordinates for the specified vertex. You must set the object's texture using ObjPrim_SetTexture beforehand.
