ObjFileB SetCharacterCode

Input

object ID (real)
character Code (real)
Sets Character Code of file. Options are:

CODE_ACP: ANSI/Shift-JIS
CODE_UTF8: UTF8
CODE_UTF16LE: UTF16 LE
CODE_UTF16BE: UTF16 BE
Default is CODE_ACP.
